字串與數字

字串(string)就是一串獨立的字串,通常做為輸出用途,例如print("hello"),看到有被引號框起來的部分,大部分都是字串。
而數字(interger整數/float非整數)不像字串大多作為輸出及標註用,它可以做運算,在輸出數字的時候不需要引號,例如print(3)。

代數

在python的代數(Variables)不只可以帶入數字,他甚至能帶入字串!例如:
pi=three point one four
print(pi)

代數的一些使用規範

數字不能作為代數,像你不能輸入17=歲數。

代數不能使用以下字串

and as assert break class continue def del elif else except exec finally for from global if import in is lambda nonlocal not or pass raise return try while with yield True False None
這些字串是python的內建語法,不能作為代數使用

len

len指令可以幫我們看出一個字串內有多少文字,例如:
x=len(hi)
print=x
就能得到2(hi有兩個字)

input

利用input我們可以跟電腦做一個簡單的互動，輸入:
name=input("what is your name")
就能被問候姓名了

特殊的*與/(乘除運算)

輸入兩個*會變成次方
a=2**3
print('a')
可以得到8
兩個'/'可以分開商與餘數,要找餘數必須用%來做為運算符號,例如:
b=2//3
print(b)
可以得到0(2/3=0...2)
而2%3可以得到2
