學到的一些東西

這兩個月的課程我們學會......交作業,當然交這個作業要稍微理解一些地方的程序,才不會打開筆記之後無從起步.

如何把倉儲的東西抓下來


先打開start弄出y槽,leo editor(要自行下載),jupyter,SciTE及cmd等,弄出一個做作業的環境
cmd會預設y槽的路徑,即Y:\>,使用cd 來選擇自己要放作業的位置,例如我要放在tmp,就輸入cd tmp,就會進到tmp
之後複製在gh-pages的github clone URL,輸入:
git clone https...........
就能把倉儲的東西複製進y槽的tmp
但打開來之後發現,只有3個檔案,這是因為我們還沒把電腦拉下來的倉儲資料切至gh-pages,欲要切換,輸入:
git checkout gh-pages
就會換成github gh-pages的檔案了
我們可以在裡面找到reveal.leo,這是我們要做作業用的檔案
下拉製作投影片,右拉上傳
每個指令碼都需要放在大於小於之間
section /section之間:個投影片
br /:換行
h數字:標題,數字為標題大小,內部建議用2 ,標題建議用1
會這些就能做一個基本的投影片了
git add -A>>打包(A=全部)
git commit -m"隨意取">>幫你打的包取個名字,方便於日後查看更新過程
如果第一次上傳或電腦有還原卡,會要求輸入信箱及使用者名稱
git push>>上傳
輸入完用戶名稱及密碼後,如果尚未更新,那可能是發生衝突,我們要先輸入git pull後 進行除錯(刪除7個一行的大小等於,vr跟v的對應)後再上傳
